# GameBlog — Веб-платформа для геймеров на Flask

Игровой блог с системой авторизации, публикацией статей и управлением профилями. 

## 🚀 Технологический стек
* **Бэкенд:** Python, Flask, Flask-Login, Flask-SQLAlchemy
* **База данных:** SQLite (легко переносится на PostgreSQL)
* **Фронтенд:** HTML5, CSS3, Jinja2 (шаблонизатор)

## 🛠 Реализованный функционал
* Полная регистрация и авторизация пользователей с хэшированием паролей.
* Система разграничения прав (просмотр доступен всем, создание постов — только авторизованным).
* Динамическое создание, чтение, обновление и удаление (CRUD) публикаций.
* Работа с реляционной базой данных через ORM.
